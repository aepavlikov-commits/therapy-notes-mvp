Формат заметки по умолчанию

Этот формат будет использоваться для всех новых сессий, если не выбрать другой вручную.

SOAP — Subjective (со слов клиента) / Objective (наблюдения) / Assessment (клиническая оценка — заполняете сами) / Plan (план). Самый распространённый формат.

DAP — Data (что происходило) / Assessment (оценка — заполняете сами) / Plan (план). Проще SOAP, без разделения на субъективное и объективное.

BIRP — Behavior (поведение и слова клиента) / Intervention (какие техники применил терапевт) / Response (реакция клиента) / Plan. Акцент на том, что сделал терапевт и как клиент отреагировал.

GIRP — Goal (цель сессии) / Intervention / Response / Plan. Похож на BIRP, но начинается с цели работы.

Свой формат — опишите словами, какие разделы и в каком порядке нужны лично вам.

Свой формат заметки

Опишите свободно, какие разделы и в каком порядке должны быть в заметке. Используется, если выбран формат "Свой формат".

Терапевтические подходы

Выберите один или несколько подходов, в которых вы работаете. Это помогает ИИ точнее отражать релевантные детали из сессии в заметке (но не добавляет интерпретаций или техник, которых не было на сессии).
